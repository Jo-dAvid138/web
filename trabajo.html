INTRODUCCION A HTML

Concepto html
El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos.
Diferencias de Html CSS y JavaScrip
La diferencia entre html,css y JavaScrip son: HTML sirve para estrucuta el codigo CSS a darle estilo y JavaScrip a agregarle lineas de codigo para hacer que la pagina web funcione adecuadamente.
Analogia con una casa Analogia con el cuerpo humano Analogia propia
En esa analogia html seria la estructura css el estilo de la casa por dentro y por fuera y JavaScrip serian los servicios como el agua y la electricidad para que el hogar sea funcional. En esta se puede tomar a html como la estructura osea o los huesos css serian las cualidades como el color de piel tipo de cabello color de ojos si es delgado o no y mas y JavaScrip serian todos nuestro sistema por ejemplo: El cerebro el sistema nervioso el sistema digestivo y todos nuestros organos para un buen funcionamiento. Pondria de ejemplo una tienda html seria los productos de la tienda css seria como esta organizada y decorada la tienda y JavaScrip los empleados que hacen que el negocio funcione como debe de ser.
